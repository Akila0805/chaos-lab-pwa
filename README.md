# Chaos Lab PWA

Chaos Lab is a **Progressive Web App (PWA)** experiment built entirely with **one HTML file**. It demonstrates how minimal code can still deliver an installable, offline-capable web app.

## ✨ Features
- **Single file simplicity** — everything in `index.html`  
- **PWA manifest** and service worker support  
- **Offline caching** for reliability  
- **Responsive design** for mobile and desktop  

## 📂 Project Structure
```
chaos-lab-pwa/
│
└── index.html   # The entire app
```

## 🚀 Getting Started
Clone the repository:
```bash
git clone https://github.com/your-username/chaos-lab-pwa.git
cd chaos-lab-pwa
```

Open `index.html` directly in your browser, or serve it locally:
```bash
# Example with Python
python -m http.server 3000
```
